# Math Algorithms


Quadratic.py:
A Quadratic equation solver, gets the zeros of a standard form equation.

commandline:
python quadratic.py a b c
ax^2 + bx + c = 0

Cubic.py:
A cubic equation solver, gets the zeros of a stand form equation.

commandline:
python quadratic.py a b c d
ax^3 + bx^2 + cx + d = 0
